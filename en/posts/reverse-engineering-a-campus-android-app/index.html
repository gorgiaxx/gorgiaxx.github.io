From this we can determine:

  • The algorithm is DESede (TripleDES) (applies DES encryption three times to each block).
  • The mode is CBC (each plaintext block is XORed with the previous ciphertext block before encryption).
  • Padding is PKCS5Padding (‚ÄúThe number of bytes to be padded equals to 8 - numberOfBytes(clearText) mod 8. So 1 to 8 bytes will be padded‚Ä¶‚Äù).
  • The initialization vector (IV) is "01234567"‚Äîin this case it effectively acts like a salt.
